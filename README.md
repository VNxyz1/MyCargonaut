## Development

Der Server kann nur auf ein build zugreifen. Jede änderung im React Code wird erst nach einem ausführen des Build Scripts gültig.
```bash
cd frontend
npm run build
```


Server Starten:
```bash
cd backend
npm run start:dev
```
